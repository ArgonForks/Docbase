Docbase

Collaborative hub for your .md files.

Getting up and running takes only 5 minutes
                        
1. npm install -g yo

2. npm install -g generator-docbase

3. yo docbase
  |
  |____ Choose an execution mode - 1) HTML 2) SPA
  |
  |____ Choose a location for your .md files
  |       1) Filesystem, 2) Github, 3) External URL
  |
  |____ Enter the path where your .md files are hosted
  |
  |____ Pick a theme color for your docs (like green, #abcdef)
  |
  |____ [Optional] Enter your github token for travis integration

4. python -m SimpleHTTPServer 1234

Docbase can read .md files from a filesystem, github repository or any http:// server.

Organize .md with ease
Docbase expects your .md documentation should confirm to a two-level layout:
1. Versioning: v1 / v2 ..., EN / BR / CN,
2. Topics: Organize .md files in folders of different topics.
It automatically reads the layout structure and renders it as a hub or you can optionally specify it in configuration.
Colorful
Docbase can adapt the docs to your preferred color theme. Specify literal color values (like golden, green, blue) or hex codes for more precision.
